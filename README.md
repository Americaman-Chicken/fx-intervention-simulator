# 為替介入インパクトシミュレーター

クロス円のFXポジションについて、円買い為替介入などで為替レートが下落した場合の評価損益、スワップ換算日数、実効レバレッジ、証拠金維持率を概算するReactアプリです。

## 主な機能

- 為替下落率を0〜10%で指定
- 複数ポジションの追加、削除、ロング/ショート切り替え
- 評価損益、ポジション総額、必要証拠金、実効レバレッジの概算
- 介入後の参考証拠金維持率とリスク目安の表示
- 1日分スワップとスワップ換算日数の表示
- スマートフォン幅に対応したレスポンシブUI

## 使用技術

- React
- Vite
- Tailwind CSS
- lucide-react

## ローカルでの起動

```bash

```


## プレビュー

```bash

```

## Vercelでの公開

Vercelでは通常、以下の設定で公開できます。

- Framework Preset: Vite
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: `dist`

特殊なリライト設定は使っていないため、現時点では `vercel.json` は不要です。

## 現時点の制限事項

- 実際の為替レートAPIとは接続していません。
- 入力値はブラウザ上のReact stateにのみ保持されます。
- 税金、手数料、スプレッド、スリッページは考慮していません。
- ロスカットルールや必要証拠金の計算方法はFX会社ごとに異なります。
- スワップポイントは日々変動します。
- Google AdSenseなどの実広告コードは未導入です。
